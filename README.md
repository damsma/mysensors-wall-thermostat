# mysensors-wall-thermostat
MySensors OLED wall thermostat

Wiring diagram:

OLED:
VCC		->	5V
GND		->  GND
DIN		->	D3
CLK     ->	D2
CS      ->	D4
DC      ->	D5
RST     ->	D6

SHT-30:
VIN     ->	3.3V
GND     ->	GND
SCL     ->	D21
SAA     ->	D20

W5100:
V+5     ->	5V
GND     ->	GND
NSS     ->	D10
MO      ->	D51
SOK     ->	D52
MI      ->	D50

Buttons:
UP		->	D17
DOWN	->	D15
MENU	->	D14
OK		->	D16
